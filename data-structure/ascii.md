## 출력 가능 아스키 문자표 (0x32 ~ 0x5f)

```javascript
var markdown = '|Dec|Hex|Character|\r\n'
markdown +=  '|--|---|--|\r\n'
for (i = 0x32; i <= 0x5f; i++) { 
    markdown += '|' + i + '|0x'+i.toString(16).padStart(2, '0') + '| ' + String.fromCharCode(i) + '|\r\n' 
}
console.log(markdown)
```
