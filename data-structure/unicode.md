## 히라가나 (89자), 일반 (2자), 상속 (2자) 
 
| 코드범위 | 코드 포인트 | 할당된 문자 |
| -------- |------------|----------------|
| U+3040..U+309F | 96         | 93               |

```javascript
var markdown = '|No|Code Point|Character|\r\n'
markdown +=  '|--|---|--|\r\n'
let sequence = 1
for (let i = 0x3040; i <= 0x309F; i++) { 
    markdown += '|' +  (sequence++) + '|U+' + i.toString(16).padStart(2, '0') + '| ' + String.fromCharCode(i) + '|\r\n' 
}
console.log(markdown)
```
